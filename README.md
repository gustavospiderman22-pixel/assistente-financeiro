# 💰 Assistente Financeiro Pessoal - SaaS

Aplicativo web completo para gestão financeira pessoal com interface moderna, dashboards profissionais e análises inteligentes.

## 🎯 Objetivo

Fornecer controle total sobre finanças pessoais, identificar gastos desnecessários, planejar metas financeiras e otimizar economias.

## ✨ Funcionalidades

- 📊 Dashboard executivo com KPIs
- 💰 Controle de receitas e despesas
- 💳 Gestão de cartões de crédito
- 🎯 Metas financeiras
- 📈 Relatórios e gráficos interativos
- 🚨 Sistema de alertas
- 🏠 Controle de patrimônio
- 💾 Persistência de dados (IndexedDB)
- 📱 Responsivo (mobile, tablet, desktop)
- 🌙 Tema escuro elegante

## 🛠️ Tecnologias

- HTML5
- CSS3 (Vanilla, sem frameworks)
- JavaScript (Vanilla ES6+)
- Chart.js (Gráficos)
- IndexedDB (Persistência local)
- Font Awesome (Ícones)

1. Clone o repositório
2. Abra `public/index.html` no navegador
3. Crie sua conta
4. Comece a registrar suas receitas e despesas
